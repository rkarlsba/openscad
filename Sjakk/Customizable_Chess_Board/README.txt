# Summary

I liked the concept of  <a href="http://www.thingiverse.com/thing:845208">Heikki_Hietala</a> but I wanted to hide the plugs.
In the same idea as I needed to resize them, I made it customizable.

Provided STLs are 30mm sides.

# Instructions

For a complete chess board game you will need to print :

-2 times "Corner Color 01"
-2 times "Corner Color 02"
-12 times "Edge Color 01"
-12 times "Edge Color 02"
-18 times "Center Color 01"
-18 times "Center Color 02"
